基于TCP的Socket和基于UDP的Socket对比

TCP是面向连接的协议，适合于对可靠性要求高的运行环境，TCP是严格的安全的。
UDP是非连续连接的通讯协议，是一钟面向无连接的协议。UDP不保证数据能够正确抵达目标,无法保证数据安全传送，多个数据包抵达目标的顺序可能和发送时顺序不同，但比它TCP更轻。也就是一般来说TCP比UDP稳定安全，UDP比TCP快速,TCP适用需要稳定安全的环境，UDP适用对稳定安全要求不大，对速度要求较高的环境。
